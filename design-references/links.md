# Reference Links

URLs to live sites you want Claude to look at for design inspiration.

## Format

One line per link, with a brief note about what to focus on:

```markdown
- https://example.com — overall layout rhythm
- https://example.com/pricing — card hover states
- https://example.com/about — typography hierarchy and color use
```
